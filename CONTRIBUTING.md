# Contributing

Thanks for your interest in improving the `webdesigner` skill.

## Contribution ideas

- refine the prompt behavior for specific industries
- add design-system variants
- improve mobile-first guidance
- add language packs
- add examples for ecommerce, beauty, SaaS, and service sites

## Guidelines

- keep instructions clear and non-redundant
- prefer real-world usefulness over verbosity
- avoid contradictory prompt rules
- keep premium brand quality as a default
- maintain mobile optimization by default
- document substantial changes in `CHANGELOG.md`

## Pull request checklist

- [ ] instructions remain coherent
- [ ] examples still match intended behavior
- [ ] wording is practical and professional
- [ ] no duplicate or conflicting rules were added
